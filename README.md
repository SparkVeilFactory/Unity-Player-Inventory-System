# Unity Player Inventory System

Clean Unity inventory system built as a lightweight and modular gameplay foundation.

## Features
- Item pickup from the world
- Inventory toggle with UI panel
- Stackable items
- Item split from stack
- Tooltip on hover
- Drop items back into the world
- Simple and easy to customize structure

## Included Systems
- Pickup interaction
- Inventory manager
- Inventory UI
- Tooltip UI
- Item database
- World item drop with physics

## Controls
- *E* = Pick up item
- *Tab* = Open / Close inventory
- *Right Click* = Split stack
- *Q* = Drop hovered item

## Notes
This project is intentionally kept clean and simple so it can be adapted to different game types and client needs.

## Project Structure
- Assets/Gameplay/System/Inventory
- Assets/Prefabs
- Assets/Art/Icons
- Assets/Art/Materials

## Use Cases
This system can be used as a base for:
- Survival games
- Farming games
- Crafting games
- RPG prototypes
- General Unity gameplay prototypes
